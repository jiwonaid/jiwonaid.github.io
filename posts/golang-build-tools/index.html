아래와 같이 간단한 golang 기반의 CLI 프로젝트를 준비합니다.

go get -u github.com/spf13/cobra

mkdir myapp
cd myapp
go mod init myapp

cobra init --pkg-name myapp

go get github.com/mitchellh/go-homedir
go get github.com/spf13/cobra
go get github.com/spf13/viper
