The Kubebuilder [Quick Start](<https://book.kubebuilder.io/quick-start.html>)

Dec. 14, 2021

Installation

brew install kubebuilder

Create a Project

mkdir guestbook
cd guestbook
kubebuilder init --domain my.domain --repo my.domain/guestbook

Create an API

kubebuilder create api --group webapp --version v1 --kind Guestbook
